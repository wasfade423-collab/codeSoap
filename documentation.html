🚢 SmartPort Gateway

Documentation officielle de l'API SOAP

Version 1.0

📌 Présentation

SmartPort Gateway est une API SOAP développée en Python permettant la gestion et le suivi des dossiers de transit portuaire. Les données sont stockées dans une base PostgreSQL hébergée sur Render.

🚀 Fonctionnalités

  • Création de dossiers de transit
  • Consultation de dossiers
  • Suivi du statut des dossiers
  • Génération automatique du WSDL
  • Gestion des réponses SOAP XML
  • Création automatique de la base de données

🏗️ Architecture du Projet

Fichier Rôle
server.py Serveur HTTP et gestion des requêtes SOAP
services.py Logique métier et traitement XML
db.py Connexion PostgreSQL et requêtes SQL
templates.py Templates XML SOAP et contrat WSDL

🗄️ Structure de la Base de Données


CREATE TABLE dossiers (
    id SERIAL PRIMARY KEY,
    reference_metier VARCHAR(255) NOT NULL UNIQUE,
    statut VARCHAR(50) NOT NULL,
    type_operation VARCHAR(100) NOT NULL
);
Champ Description
id Identifiant unique
reference_metier Référence métier du dossier
statut État actuel du dossier
type_operation Type d'opération portuaire

📄 Accès au WSDL


GET /?wsdl

Cette route permet aux clients SOAP de récupérer automatiquement le contrat WSDL du service.

🔗 Services SOAP

CreateDossier

Création d'un nouveau dossier de transit.

Paramètre Type
reference_metier String
statut String
type_operation String

GetDossier

Consultation d'un dossier existant.

Paramètre Type
reference_metier String

📬 Exemple de Requête SOAP


<soapenv:Envelope
xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/">

    <soapenv:Body>

        <CreateDossierRequest>

            <reference_metier>REF001</reference_metier>
            <statut>EN_COURS</statut>
            <type_operation>IMPORT</type_operation>

        </CreateDossierRequest>

    </soapenv:Body>

</soapenv:Envelope>

📤 Exemple de Réponse SOAP


<soapenv:Envelope
xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/">

    <soapenv:Body>

        <CreateDossierResponse>

            <message>
                Dossier créé avec succès
            </message>

        </CreateDossierResponse>

    </soapenv:Body>

</soapenv:Envelope>

🛡️ Gestion des Erreurs

  • Référence métier déjà existante
  • Données XML invalides
  • Erreur PostgreSQL
  • Requête SOAP mal formée
  • Erreur interne du serveur
